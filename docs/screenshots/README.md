# Screenshots

Drop image files here and they render in the main [README](../../README.md).

Recommended shots (the README already references these filenames):

| File | What to capture |
|------|-----------------|
| `dashboard.png` | The dashboard in **light** mode — hero header, stat tiles, budget meters, bulletin board. |
| `expenses.png` | The **Expenses** tab — the new-expense form, the settle-up panel ("who owes whom"), and the month's expense table. |
| `mobile-dark.png` | The dashboard on a narrow (mobile) viewport in **dark** mode, showing the bottom navigation bar. |
| `demo.gif` | A ~10s screen recording: add an expense → open settle-up → resolve a recipe into the shopping list. |

Tips:
- Use the built-in demo flow or a throwaway household so no real personal data
  is published.
- Keep PNGs reasonably sized (≈ 1000px wide); optimise the GIF so it stays small.
